使用html5 canvas元素和javascript配合使用在页面上绘制图形，Canvas本身没有绘图能力， 绘制工作都是在javascript中完成

  1. 没有 src 和 alt 属性，只有两个属性 width 和 height

    <canvas id="tutorial" width="150" height="150"></canvas>

  2. 由于某些较老的浏览器（尤其是IE9之前的IE浏览器）或者文本浏览器不支持HTML元素”canvas”，所以可以在canvas元素里面设置替换内容

  1. 闭合标签不可省略
  2. 通过getContext() 的方法来获得渲染上下文和它的绘画功能，getContext()只有一个参数，那就是上下文的格式。

2D绘制类型：

  1. 矩形
    以下是4个绘制矩形位图的方法。
    1. CanvasRenderingContext2D.clearRect(x, y, width, height)
      设置指定矩形区域内（以 点 (x, y) 为起点，范围是(width, height) ）所有像素变成透明，并擦除之前绘制的所有内容。
    2. CanvasRenderingContext2D.fillRect(x, y, width, height)
      绘制填充矩形，矩形的起点在 (x, y) 位置，矩形的尺寸是 width 和 height 。
    3. CanvasRenderingContext2D.strokeRect(x, y, width, height)
      在 canvas 中，使用当前的绘画样式，描绘一个起点在 (x, y) 、宽度为 w 、高度为 h 的矩形。
    4. rect(x, y, width, height)
      绘制一个左上角坐标为（x,y），宽高为width以及height的矩形，当该方法执行的时候，moveTo()方法自动设置坐标参数（0,0）。也就是说，当前笔触自动重置会默认坐标。

示例：

  1. 文本
    下面是3个绘制文本的方法。

    1. CanvasRenderingContext2D.fillText()
      在(x,y)位置绘制（填充）文本。
    2. CanvasRenderingContext2D.strokeText()
      在(x,y)位置绘制（描边）文本。
    3. CanvasRenderingContext2D.measureText()
      返回 TextMetrics 对象。

  2. 线型
    下面的方法和属性控制如何绘制线。

    1. CanvasRenderingContext2D.lineWidth
      线的宽度。默认 1.0
    2. CanvasRenderingContext2D.lineCap
      线末端的类型。 允许的值： butt (默认), round, square.
    3. CanvasRenderingContext2D.lineJoin
      定义两线相交拐点的类型。允许的值：round, bevel, miter(默认)。
    4. CanvasRenderingContext2D.miterLimit
      斜接面限制比例。默认 10。
    5. CanvasRenderingContext2D.getLineDash()
      返回当前线段样式的数组，数组包含一组数量为偶数的非负数数字。
    6. CanvasRenderingContext2D.setLineDash()
      设置当前的线段样式。
    7. CanvasRenderingContext2D.lineDashOffset
      描述在哪里开始绘制线段。

  3. 文本样式
    下面的属性控制如何设计文本。

    1. CanvasRenderingContext2D.font
      字体设置。 默认值 10px sans-serif。
    2. CanvasRenderingContext2D.textAlign
      文本对齐设置。 允许的值： start (默认), end, left, right 或 center.
    3. CanvasRenderingContext2D.textBaseline
      基线对齐设置。 允许的值： top, hanging, middle, alphabetic (默认),ideographic, bottom.
    4. CanvasRenderingContext2D.direction
      文本的方向。 允许的值： ltr, rtl, inherit (默认).

  4. 填充和描边样式
    填充设计用于图形内部的颜色和样式，描边设计用于图形的边线。

    1. CanvasRenderingContext2D.fillStyle
      图形内部的颜色和样式。 默认 #000 (黑色).
    2. CanvasRenderingContext2D.strokeStyle
      图形边线的颜色和样式。 默认 #000 (黑色).

  5. 渐变和图案

    1. CanvasRenderingContext2D.createLinearGradient()
      创建一个沿着参数坐标指定的线的线性渐变。
    2. CanvasRenderingContext2D.createRadialGradient()
      创建一个沿着参数坐标指定的线的放射性性渐变。
    3. CanvasRenderingContext2D.createPattern()
      使用指定的图片 (a CanvasImageSource)创建图案。通过 repetition 变量指定的方向上重复源图片。此方法返回 CanvasPattern对象。

  6. 阴影

    1. CanvasRenderingContext2D.shadowBlur
      描述模糊效果。 默认 0
    2. CanvasRenderingContext2D.shadowColor
      阴影的颜色。 默认fully-transparent black.
    3. CanvasRenderingContext2D.shadowOffsetX
      阴影水平方向的偏移量。 默认 0.
    4. CanvasRenderingContext2D.shadowOffsetY
      阴影垂直方向的偏移量。 默认 0.

  7. 路径
    下面的方法用来操作对象的路径。

    1. CanvasRenderingContext2D.beginPath()
      清空子路径列表开始一个新的路径。当你想创建一个新的路径时，调用此方法。
    2. CanvasRenderingContext2D.closePath()
      使笔点返回到当前子路径的起始点。它尝试从当前点到起始点绘制一条直线。如果图形已经是封闭的或者只有一个点，那么此方法不会做任何操作。
    3. CanvasRenderingContext2D.moveTo(x, y)
      将一个新的子路径的起始点移动到(x，y)坐标。
    4. CanvasRenderingContext2D.lineTo(x, y)
      使用直线连接子路径的最后的点到x,y坐标。
    5. CanvasRenderingContext2D.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)
      添加一个3次贝赛尔曲线路径。该方法需要三个点。 第一、第二个点是控制点，第三个点是结束点。起始点是当前路径的最后一个点，绘制贝赛尔曲线前，可以通过调用 moveTo() 进行修改。
      cp1x,cp1y为控制点一，cp2x,cp2y为控制点二，x,y为结束点
    6. CanvasRenderingContext2D.quadraticCurveTo(cp1x, cp1y, x, y)
      绘制二次贝塞尔曲线，cp1x,cp1y为一个控制点，x,y为结束点

    7. CanvasRenderingContext2D.arc(x, y, radius, startAngle, endAngle, anticlockwise)
      绘制一段圆弧路径， 圆弧路径的圆心在 (x, y) 位置，半径为 r ，根据anticlockwise （默认为顺时针）指定的方向从 startAngle 开始绘制，到 endAngle 结束。
      arc()函数中的角度单位是弧度，不是度数。角度与弧度的js表达式:$radians=(Math.PI/180)*degrees。$

    8. CanvasRenderingContext2D.arcTo(x1, y1, x2, y2, radius)
      根据控制点和半径绘制圆弧路径，使用直线连接前一个点。

    9. CanvasRenderingContext2D.ellipse()
      添加一个椭圆路径，椭圆的圆心在（x,y）位置，半径分别是radiusX 和 radiusY ，按照 anticlockwise （默认顺时针）指定的方向，从 startAngle 开始绘制，到 endAngle 结束。
    10. CanvasRenderingContext2D.rect()
      创建一个矩形路径，矩形的起点位置是 (x, y) ，尺寸为 width 和 height。

    示例：

    1. 三角形

    2. 笑脸

    3. 双三角形

    4. 圆（弧）

    5. 二次贝塞尔曲线 （对话冒泡）

    6. 三次贝塞尔曲线（心形）

    7. 组合使用

  8. 绘制路径

    1. CanvasRenderingContext2D.fill()
      使用当前的样式填充子路径。
    2. CanvasRenderingContext2D.stroke()
      使用当前的样式描边子路径。
    3. CanvasRenderingContext2D.drawFocusIfNeeded()
      如果给定的元素获取了焦点，那么此方法会在当前的路径绘制一个焦点。
    4. CanvasRenderingContext2D.scrollPathIntoView()
      将当前或给定的路径滚动到窗口。
    5. CanvasRenderingContext2D.clip()
      从当前路径创建一个剪切路径。在 clip() 调用之后，绘制的所有信息只会出现在剪切路径内部。例如： 参见 Canvas教程中的 剪切路径 。
    6. CanvasRenderingContext2D.isPointInPath()
      判断当前路径是否包含检测点。
    7. CanvasRenderingContext2D.isPointInStroke()
      判断检测点是否在路径的描边线上。

  9. 变换
    在 CanvasRenderingContext2D 渲染背景中的对象会有一个当前的变换矩阵，一些方法可以对其进行控制。当创建当前的默认路径，绘制文本、图形和Path2D对象的时候，会应用此变换矩阵。下面列出的方法保持历史和兼容性的原因，是为了SVGMatrix对象现在能够应用于大部分 API ，将来会被替换。

    1. CanvasRenderingContext2D.currentTransform
      当前的变换矩阵 (SVGMatrix 对象)。
    2. CanvasRenderingContext2D.rotate()
      在变换矩阵中增加旋转，角度变量表示一个顺时针旋转角度并且用弧度表示。
    3. CanvasRenderingContext2D.scale()
      根据 x 水平方向和 y 垂直方向，为canvas 单位添加缩放变换。
    4. CanvasRenderingContext2D.translate()
      通过在网格中移动 canvas 和 canvas 原点 x 水平方向、原点 y 垂直方向，添加平移变换
    5. CanvasRenderingContext2D.transform()
      使用方法参数描述的矩阵多次叠加当前的变换矩阵。
    6. CanvasRenderingContext2D.setTransform()
      重新设置当前的变换为单位矩阵，并使用同样的变量调用 transform() 方法。
    7. CanvasRenderingContext2D.resetTransform()
      使用单位矩阵重新设置当前的变换。

  10. 合成

    1. CanvasRenderingContext2D.globalAlpha
      在合成到 canvas 之前，设置图形和图像透明度的值。默认 1.0 (不透明)。
    2. CanvasRenderingContext2D.globalCompositeOperation
      通过 globalAlpha 应用，设置如何在已经存在的位图上绘制图形和图像。
      绘制图像
    3. CanvasRenderingContext2D.drawImage()
      绘制指定的图片。该方法有多种格式，提供了很大的使用灵活性。

  11. 像素控制

    1. CanvasRenderingContext2D.createImageData()
      使用指定的尺寸，创建一个新的、空白的 ImageData 对象。所有的像素在新对象中都是透明的。
    2. CanvasRenderingContext2D.getImageData()
      返回一个 ImageData 对象，用来描述canvas区域隐含的像素数据，这个区域通过矩形表示，起始点为(sx, sy)、宽为sw、高为sh。
    3. CanvasRenderingContext2D.putImageData()
      将数据从已有的 ImageData 绘制到位图上。 如果提供了脏矩形，只能绘制矩形的像素。

  12. 图像平滑

    1. CanvasRenderingContext2D.imageSmoothingEnabled
      图像平滑的方式；如果禁用，缩放时，图像不会被平滑处理。

  13. canvas 状态
    CanvasRenderingContext2D渲染环境包含了多种绘图的样式状态（属性有线的样式、填充样式、阴影样式、文本样式）。下面的方法会帮助你使用这些状态：

    1. CanvasRenderingContext2D.save()
      使用栈保存当前的绘画样式状态，你可以使用 restore() 恢复任何改变。
    2. CanvasRenderingContext2D.restore()
      恢复到最近的绘制样式状态，此状态是通过 save() 保存到”状态栈“中最新的元素。
    3. CanvasRenderingContext2D.canvas
      对 HTMLCanvasElement 只读的反向引用。如果和 <canvas> 元素没有联系，可能为null。

  14. 点击区域

    1. CanvasRenderingContext2D.addHitRegion()
      给 canvas 添加点击区域。
    2. CanvasRenderingContext2D.removeHitRegion()
      从 canvas 中删除指定 id 的点击区域。
    3. CanvasRenderingContext2D.clearHitRegions()
      从 canvas 中删除所有的点击区域
